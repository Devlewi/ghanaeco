# 🌍 GHANA ECO – Site d’Actualités Économiques et Business au GHANA

**Domaine** : [GHANA-ECO.COM](https://GHANA-ECO.COM)  
**Frontend** : Next.js + Tailwind CSS  
**Backend (CMS)** : WordPress (Headless via REST API)  
**Internationalisation** : Intégrée avec Next.js

---

## 📰 Présentation

**GHANA ECO** est un site web professionnel dédié à l’**actualité économique, business et financière** au GHANA et en Afrique.  
Le projet repose sur une architecture moderne : un frontend développé avec **Next.js** et **Tailwind CSS**, et un backend WordPress utilisé en **mode headless** pour la gestion des contenus via l’API REST.

Le site propose une navigation rapide, multilingue, mobile-first et optimisée SEO.

---

## ✨ Fonctionnalités clés

- 📄 **Affichage dynamique des articles via l’API WordPress**
- 🗂️ **Catégorisation des articles (économie, finance, entrepreneuriat, etc.)**
- 🌐 **Internationalisation multilingue (français, anglais)**
- 🔎 **Recherche intégrée**
- 📱 **Responsive design pour une lecture optimisée sur tous les appareils**
- 🔗 **Partage social, SEO optimisé, et navigation rapide...**
